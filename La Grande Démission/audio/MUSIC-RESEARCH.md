## 6. Guide par usage podcast

| Usage | Tracks recommandees | Pourquoi |
|-------|---------------------|----------|
| **Intro / generique** | Stomps & Claps, Happy Hustle, Funk Forward* | Energie immediate, reconnaissable |
| **Transition** | Feel Good 50s*, Funky Funk Short 30s | Courts, groove, pas envahissants |
| **Fond sous la voix** | Strut Effect, Funky Groove BG* | Assez discrets pour laisser la parole |
| **Outro** | Celebration*, Joyful, The Day Felt Easy* | Finir positif, donner envie de revenir |
| **Moments forts** | Happy Ukulele Claps*, Freccero - Get Up | Ponctuer une idee avec de l'emotion |
| **Humour** | I Got a Stick, Monkeys Spinning Monkeys* | Anecdotes droles, legerete |

## 8. Methode de telechargement

```bash
# yt-dlp avec impersonate bypass Cloudflare (~50% de reussite)
yt-dlp --extractor-args "generic:impersonate" \
  --extract-audio --audio-format mp3 \
  -o "%(title)s.%(ext)s" \
  "URL_PIXABAY"
```

Cloudflare rate-limit apres ~6 requetes consecutives. Espacer les telechargements ou passer par le browser pour les tracks bloquees.
