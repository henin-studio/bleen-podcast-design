# Questionnaire de Brief — Design de Podcast

> Template réutilisable pour tout projet de podcast design chez Bleen.
> À utiliser EN CONVERSATION, pas comme formulaire à remplir.
> Les 5 premières minutes, le client dit ce qu'il pense vouloir.
> Les 20 suivantes, grâce à ces questions, il dit ce qu'il veut vraiment.

## Comment utiliser ce questionnaire

1. **En conversation**, pas en formulaire envoyé par email
2. L'ordre n'est pas fixe — tirer le fil de ce que le client dit
3. Les questions 37-43 (sensorielles) débloquent souvent les gens coincés dans le rationnel
4. Les questions 9-11 (rejet) sont plus fiables que les questions d'inspiration
5. La question 50 est le brief. Tout le reste prépare cette réponse.

**Le brief n'est pas dans les réponses individuelles. Il est dans les patterns qui émergent entre les réponses.**
