### 1.4. Regles d'utilisation

**Ratio 60-30-10 :**
- **60%** — Velotaf Red-Orange (`#E94B35`) ou fond clair (`#FAF6F2`) selon le contexte
- **30%** — Blanc (`#FFFFFF`) pour la typo et l'air
- **10%** — Bleu marine (`#2E4057`) ou accents fonctionnels

**Contraste :**
- Blanc sur orange signature → ratio ~4.2:1 (AA pour gros texte, pas AA pour petit texte)
- Pour du petit texte sur orange, utiliser `#FFFFFF` en gras ou passer sur fond sombre
- Noir doux (`#2D2D2D`) sur creme (`#FAF6F2`) → ratio ~14:1 (AAA)

**Interdits :**
- Pas de gris moyen (#888, #999) — la palette est franche
- Pas de pastels — toujours saturation > 30%
- Pas de noir pur (#000000) — utiliser `#2D2D2D` ou `#1A1A1A`
- Pas de bleu electrique ou vert fluo — le bleu reste marine/ardoise

## 2. Typographie

### Logo "VELOTAF"

| Propriete | Valeur |
|-----------|--------|
| Style | Manuscrite / brush, tout en majuscules |
| Graisse | Bold, traits epais irreguliers |
| Couleur | Blanc pur sur fond orange |
| Caractere | Energique, decontracte, fait-main |
| Accent | Le "E" de "VELOTAF" porte un accent aigu (É) -- important, ne pas l'oublier |
| Inclinaison | Legere inclinaison vers la droite, dynamique |
| Taille relative | Le titre occupe ~30% de la largeur en haut de la cover |

### Famille de police recommandee

Pour decliner l'identite au-dela du logo :

| Usage | Police suggeree | Alternative | Justification |
|-------|-----------------|-------------|---------------|
| **Logo / Titres** | Brush script custom (ou Permanent Marker, Archivo Black) | Bebas Neue, Oswald | Energie, caractere urbain |
| **Sous-titres** | Montserrat Bold | Poppins Bold | Lisibilite, modernite |
| **Corps de texte** | Montserrat Regular | Source Sans Pro | Neutralite, confort de lecture |
| **Accents / citations** | Caveat, Patrick Hand | Kalam | Rappel du cote manuscrit du logo |

### Hierarchie typographique

```
TITRE (logo style)     -- Brush/manuscrit, blanc, 48-72pt
Sous-titre             -- Sans-serif bold, blanc ou noir, 24-32pt
Corps                  -- Sans-serif regular, 14-18pt
Legende / meta         -- Sans-serif light, 10-12pt
```

## 3. Illustration

### Style

| Propriete | Description |
|-----------|-------------|
| Type | Illustration vectorielle flat design |
| Rendu | Semi-realiste, stylise (pas cartoon, pas photo-realiste) |
| Traits | Contours nets, pas de ligne noire visible (formes de couleur pleines) |
| Ombres | Minimales, par aplats de couleur plus sombre (pas de degradé doux) |
| Details | Suffisamment detailles pour etre reconnaissable (lunettes, barbe, blazer) mais simplifies |
| Perspective | Vue de face legerement en contre-plongee, dynamique |

### Personnage

- Homme barbu a lunettes, style urbain-chic (blazer + chemise)
- Expression souriante, decontractee
- Position : sur un velo, mains sur le guidon, en mouvement
- Le personnage est **l'element central** -- il occupe ~60% de la hauteur

### Principes pour decliner

- Garder le style **flat avec aplats de couleur**
- Pas de textures, pas de bruit, pas d'effets photo
- Les illustrations doivent rester **chaleureuses et accessibles**
- Le velo doit toujours etre present comme element iconique
- Le personnage peut etre decline en differentes situations (pedalant, arrivant au bureau, sous la pluie, etc.)

## 4. Composition & mise en page

### Cover art (format carre ~3000x3000px)

```
+------------------------------------------+
|                                          |
|    V É L O T A F                         |
|    (typo brush, blanc, haut)             |
|                                          |
|         [PERSONNAGE SUR VELO]            |
|         (centre, occupe 60%              |
|          de la hauteur)                  |
|                                          |
|                                          |
|                               [logo] bas |
+------------------------------------------+
        FOND : orange-rouge uni
```

### Principes de composition

- **Titre en haut** : toujours au-dessus du personnage, jamais derriere
- **Personnage centre** : point focal, regard vers le spectateur
- **Fond uni** : pas de photo, pas de paysage. La couleur orange suffit
- **Espacement genereux** : le design respire, pas d'encombrement
- **Logo plateforme** : petit, discret, coin bas-droit

### Grille

- Marges : ~8% de chaque cote
- Le titre demarre dans le tiers superieur
- Le personnage est centre horizontalement, ancre au tiers inferieur
- Ratio illustration/fond : ~55/45 (le fond orange reste bien visible)

## 7. Do's & Don'ts

### Do's

- Toujours utiliser le fond orange comme element dominant
- Garder la typo manuscrite pour le logo uniquement (pas pour le corps de texte)
- Decliner l'illustration dans le meme style flat
- Maintenir le contraste blanc/orange pour la lisibilite
- Garder l'ambiance decontractee et positive

### Don'ts

- Ne pas utiliser de photos en fond (le flat design EST l'identite)
- Ne pas ajouter de degradés complexes ou d'effets 3D
- Ne pas changer la couleur du fond signature (pas de bleu, pas de vert)
- Ne pas ecrire le logo en minuscules ou dans une police standard
- Ne pas surcharger les visuels -- la simplicite est la force
- Ne pas utiliser de noir pur (#000000) -- preferer le noir doux (#2D2D2D)
